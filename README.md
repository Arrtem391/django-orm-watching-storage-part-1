# Пульт охраны банка

Это внутренний репозиторий для сотрудников банка «Сияние». Если вы попали в этот репозиторий случайно, то вы не сможете его запустить, т.к. у вас нет доступа к БД, но можете свободно использовать код вёрстки или посмотреть как реализованы запросы к БД.

Пульт охраны — это сайт, который можно подключить к удалённой базе данных с визитами и карточками пропуска сотрудников нашего банка.

### Как установить

Python3 должен быть уже установлен. Затем используйте `pip` (или `pip3`, если есть конфликт с Python2) для установки зависимостей:

```pip install -r requirements.txt```

### Переменные окружения

Перед запуском программы необходимо настроить следующие переменные окружения. Они используются для подключения к базе данных и настройки Django:

| Переменная | Описание | Пример значения |
|------------|-----------|-----------------|
| `DJANGO_SETTINGS_MODULE` | Указывает Django на модуль с настройками проекта | `settings` |
| `DATABASE_URL` | URL для подключения к PostgreSQL базе данных | `postgresql://user:password@host:port/dbname` |
| `SECRET_KEY` | Секретный ключ Django для криптографических операций | `your-secret-key-here` |
| `DEBUG` | Режим отладки Django (`True`/`False`) | `False` |
| `ALLOWED_HOSTS` | Разрешённые хосты для Django приложения | `localhost,127.0.0.1,.example.com` |

### Как использовать

Скрипт `main.py` подключается к базе данных банка и выводит статистику по пропускам сотрудников.

### Пример вывода в консоль:

```
Количество пропусков: 125
Количество активных пропусков: 89
```

В этом примере:
- Всего в базе данных зарегистрировано 125 пропусков
- Из них активно (не заблокированы) 89 пропусков
